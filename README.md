- Swift 4

## 系统要求

## 所以我怎么用呢？
上架啥的根本没戏完全不可能 (=・ω・=)

不过你要是想尝试这个 App 的话有两种方法：
1. 你可以发邮件告诉我你的 Apple ID，成为 BilibiliCD 的测试员。不过名额有限，现在还有 15 个名额。
2. 如果你有技术的话推荐直接把代码下载下来然后装到手机里，不过在编译前注意需要修改 App Group 等信息。
